Install Claude
On Your Windows PC

Two ways to get Claude running locally — the desktop app for everyone, and Claude Code for developers. Pick your path and follow the steps.

Windows 10 or 11 x64 & ARM64 supported Under 2 minutes Free account works

Step 1 Go to the official download page
BrowserStart here

Open claude.ai/download

Always download Claude directly from Anthropic's official page. Do not use third-party stores or Microsoft Store versions — they may lack features like Cowork.

Your move: Open your browser and navigate to claude.ai/download. You'll see buttons for macOS, Windows, and Windows (ARM64).
Step 2 Download the Windows installer
Download.exe file

Click the Windows button

Click Windows to download the standard installer. If you have an ARM-based device (e.g. Snapdragon X Elite), choose Windows (arm64) instead.

Your move: The .exe file will appear in your Downloads folder when complete. This usually takes under 30 seconds.
Step 3 Run the installer
Install~1 min

Double-click the .exe file

Open your Downloads folder and double-click the installer. Windows may ask for administrator permission — click Yes to proceed. The installer runs silently with no options to configure.

SmartScreen warning? If Windows shows a SmartScreen alert, click More info → Run anyway. This is normal for newly released apps from Anthropic.
Step 4 Sign in and start using Claude
LaunchSign in

Claude opens automatically

Once installed, Claude launches and asks you to sign in. Use the same account you use on claude.ai — either Google or email/password. No account yet? Create a free one at claude.ai/signup.

Your move: Find Claude in your Start menu any time. Pin it to your taskbar for quick access during the bootcamp sessions.
You're set: Claude Desktop is now installed. Chat is free for all accounts. Cowork (autonomous agent) and Claude Code require a paid plan.

Step 1 Open PowerShell as Administrator
PowerShellAdmin

Launch PowerShell with admin rights

Press the Windows key, type PowerShell, right-click it in the results, and choose Run as administrator. Click Yes when prompted.

Your move: You should see a blue terminal window titled "Administrator: Windows PowerShell". You're ready for the next step.
Step 2 Run the native installer command
CommandPaste & run

Paste this into PowerShell and press Enter

This downloads the Claude Code binary, places it in your PATH, and sets up automatic updates — all in under a minute. No Node.js needed.

iwr https://claude.ai/install.ps1 | iex
What this does: Downloads the installer script and executes it. Claude Code is placed at ~\.local\bin and your PATH is updated automatically.
Step 3 Close and reopen PowerShell
RestartRequired

Close the terminal window entirely

The PATH update doesn't take effect in the current session. Close PowerShell completely, then open a new one (standard, no need for admin this time).

Step 4 Sign in and run Claude
LaunchFirst run

Type claude and press Enter

A browser window opens automatically for OAuth authentication. Sign in with your Anthropic account — no API key management needed. After signing in, return to the terminal.

claude
From any project folder: Navigate into your project with cd C:\Users\You\Projects\my-app then run claude to start a session inside that codebase.
Claude Code is live. Navigate into any project folder and type claude to start. It reads your files and lets you make changes with plain English.
"claude is not recognized" error? Run this in a new PowerShell window, then close and reopen it:
[Environment]::SetEnvironmentVariable("PATH", "$env:PATH;$env:USERPROFILE\.local\bin", [EnvironmentVariableTarget]::User)

Step 1 Install Node.js 18 or later
Node.jsPrerequisite

Download from nodejs.org (LTS recommended)

Go to nodejs.org and download the LTS version. Run the installer with default settings. After installing, verify it worked by opening PowerShell and running:

node --version
You should see: Something like v22.0.0. If the version is below 18, download a newer LTS from nodejs.org and reinstall.
Step 2 Install Claude Code globally
npmGlobal install

Open PowerShell as Administrator and run:

The -g flag installs Claude Code globally so you can run it from any folder on your system. This takes about a minute.

npm install -g @anthropic-ai/claude-code
Permission error? Run PowerShell as Administrator (right-click → Run as administrator) and retry. Do not use sudo on Windows — it won't help and can break things.
Step 3 Sign in via browser
LaunchFirst run

Run claude to open the browser login

A browser window opens automatically. Sign in with your Anthropic account. After signing in, return to the terminal — you're ready to work.

claude
From any project: Navigate with cd into a project folder, then type claude. It reads your codebase and takes instructions in plain English.
Claude Code is installed via npm. To keep it updated, run npm install -g @anthropic-ai/claude-code periodically, or switch to the native installer for automatic updates.
